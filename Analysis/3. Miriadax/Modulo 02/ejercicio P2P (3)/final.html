Los caracteres especiales en HTML5 son: toda clase de elementos propios de un lenguaje el cual no pueden ser recibidos de forma directa por el navegador; en esos casos se suele aplicar una etiqueta que avale el uso de estos caracteres o la adición de estos mediante un código único de cada caracter especial; algunos ejemplos son la ñ, la tílde, la diéresis (Ü) entre otros.
